In addition to this, ambiguous columns error can also occur this is due to the columns having the same value and Oracle being confused in which column is wanting to be retrieved. In order to correct this duplicate column must state the table name throughout the query.

Week 9

31/03/20

The learning objectives for this week are to appreciate the importance of security and integrity in databases, the different types of integrity control, common threats, authentication and access to databases.

Security refers to the protection of data; ensuring that control are put in place to stop unauthorised access and transactions. Integrity refers to the accuracy or validity of the data; introducing controls to ensure activities users are trying to undertake leave the database in a correct state. There are three types of building integrity into a database: Entity, referential and general data. There is one simple rule of entity integrity, no column of a primary key can contain a null and every value of a primary key is unique. In addition referential is where a foreign key must not contain any unmatched values and general data is where constraints are applied to assist the maintenance of the integrity these include, Unique, not null and check.
